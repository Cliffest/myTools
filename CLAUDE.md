# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Email Notification

When there is `#email` mark at the beginning of user prompt, signifies that you should run in "experiment notification with email" mode--follow the rules in `docs/instructions.email.md`. If there is NO `#email` mark, DISMISS this whole part; if user ask you to send email but without the `#email` mark, ask for second confirmation. 

## Commit Messages

When generating `git commit` messages, follow the Bilingual Conventional-Commit rules in `docs/instructions.commit-style.md`.
